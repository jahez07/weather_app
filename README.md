# Weather Application - PM Accelorator Program

## How to run the program. 
1. Clone the repository 
2. Open in editor (VSCode / Cursor)
3. Run <br>
&nbsp; 1. ```npm install``` <br>
&nbsp; 2. ```npm run dev``` <br>

## Technologies Used
1. Vite
2. React 
3. openweathermap API
